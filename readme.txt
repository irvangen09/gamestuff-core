A fast, lightweight, and maintainable Gutenberg block foundation for WordPress.

== Description ==

GameStuff Core is a Gutenberg block foundation that prioritizes quality, 
performance, and native integration with the WordPress ecosystem.
This plugin is an evolution of LunarBlocks and GameStuff Block,
unified into a single, cleaner, and more consistent architecture.

Available blocks:

* **GameStuff TOC** — Dynamic Table of Contents for articles and documentation.

== Installation ==

This plugin is currently under early development and is not yet ready for 
production environments.

== Changelog ==

= 0.1.0 =
* Initialize GameStuff Core as the new foundation, 
unifying the architectures of LunarBlocks and GameStuff Block.
* Add: GameStuff TOC block — a dynamic table of contents 
featuring hierarchical view, auto insert, smooth scroll, 
active heading highlighting, collapsible subheadings, 
per-device sticky positioning, and numeration options (None/Decimal/Roman).
